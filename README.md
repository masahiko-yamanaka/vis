# Vis - Very Important Sounder

## 概要
目の前の箱を世界中の人々の心を動かす楽器にする。

## ビルドと実行方法

このプロジェクトはRustで書かれています。
実行するには、まずRustのツールチェインがインストールされていることを確認してください（[rustup](https://rustup.rs/) を推奨）。

プロジェクトのルートディレクトリで、以下のコマンドを実行します。

```bash
cargo build
cargo run
```

## 使い方 (現在のテスト段階の機能)

**注意:** 以下のキーマッピングは、現在のテスト段階におけるものです。将来のバージョンでは変更される可能性があります。

アプリケーションが起動すると、ターミナルに以下の案内が表示されます。

```
--- Vis: Chord & Percussion Mode ---
1〜9: 単音 / V, B, N: 和音(C, G, Am) / Space: シンバル
```

*   `1`〜`9`キー (実装例として`1`キーのみ機能):
    *   `1`: 単音を再生します。
*   `V`キー: Cメジャーの和音を再生します。
*   `B`キー: Gメジャーの和音を再生します。
*   `N`キー: Aマイナーの和音を再生します。
*   `Space`キー: シンバル音を再生します。
*   `Escape`キー: アプリケーションを終了します。
